## OpenTCM: Bridging Traditional Chinese Medicine with AI

This project develops a GraphRAG-empowered Large Language Model system for Traditional Chinese Medicine knowledge retrieval and diagnosis assistance.

### Key Features
- Knowledge graph construction from TCM literature
- GraphRAG integration with LLM
- Interactive Q&A system
- Clinical decision support
