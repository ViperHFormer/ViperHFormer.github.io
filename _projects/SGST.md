## SGST: Machine Learning Approach for Cavitation Fault Diagnosis

Developed during my exchange at NTU Singapore, this project applies machine learning techniques to diagnose cavitation faults in industrial machinery.

### Methodology
- Short-time Fourier Transform for feature extraction
- GAN-based data augmentation
- Swin Transformer for image classification

### Achievements
- Published at **ICCCN 2023**
- National Invention Patent (Oct 2023)
- Applied to real-world industrial datasets

### Technologies
- Python, TensorFlow
- Signal Processing
- Computer Vision
